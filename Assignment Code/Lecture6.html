HTML Forms - Lecture 6

1. Summary

HTML Forms are used to collect user input in web applications. Forms are commonly used for login, registration, and feedback. The data entered by the user is sent to the server for processing.

2. Covered Topics

  • Introduction to HTML Forms
  • <form> element
  • Form attributes (action, method)
  • <label> and <input> elements
  • Common input types (text, email, password)
  • Checkbox and radio buttons
  • Dropdown lists using <select>
  • <textarea> for multi-line text
  • <button> element
  • Form validation attributes
  • Placeholder, name, and value attributes
  • Form submission process

3. Basic Form Example

<form action="/submit" method="post">

Name:
<input type="text" name="username">

Email:
<input type="email" name="email">

Password:
<input type="password" name="password">

<button type="submit">Submit</button>

</form>

4. Checkbox Example

<input type="checkbox" name="subscribe"> Subscribe to newsletter

5. Radio Button Example

Gender:
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

6. Dropdown Example

<select name="country">
<option>Pakistan</option>
<option>UK</option>
<option>USA</option>
</select>

7. Textarea Example

<textarea rows="4" cols="40">
Enter your message here
</textarea>

8. Form Validation Example

<input type="text" name="username" required maxlength="10" placeholder="Enter username">

9. Learning Outcomes

  • HTML forms are used to collect user input.
  • Forms include elements like input, textarea, select, and button.
  • Validation attributes help control user input.
  • Forms send data to the server using GET or POST method.
