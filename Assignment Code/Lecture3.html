Covered Topics

Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create webpages. It defines the structure of a webpage using tags and elements.

History of HTML

HTML was invented by Tim Berners-Lee in 1989. Over time it evolved from HTML 1.0 to modern HTML5 which supports multimedia and better webpage structure.

HTML vs XHTML

HTML is flexible and allows some tags to remain unclosed. XHTML is stricter and requires proper nesting and closing of all tags.

Block and Inline Elements

Block elements start on a new line and take full width such as div, p, and h1. Inline elements stay in the same line like span, img, and a.

Example Code

Basic HTML Example

<!DOCTYPE html>
<html>
<head>
<title>My Page</title>
</head>
<body>
<h1>Welcome</h1>
<p>This is my first webpage</p>
</body>
</html>

Summary

HTML is the basic language used to build webpages. It uses tags and elements to structure web content. HTML has evolved over time and modern HTML5 supports many advanced features. Understanding HTML tags, attributes, and the difference between block and inline elements helps developers create well-structured webpages.
