HTML5 Introduction - Lecture Summary

1. What is HTML5?

HTML5 is the latest version of HTML used to create modern websites. It provides new elements, better structure, multimedia support, and works on different devices.

2. Key Features of HTML5

  • Responsive Design for different screen sizes
  • Viewport Meta Tag for mobile devices
  • Built-in Multimedia support (Audio & Video)
  • Cross-platform compatibility
  • New form input types (email, date, number etc.)
  • Offline support using localStorage

3. HTML5 vs HTML4

  • HTML4: Mostly used non-semantic tags like div and span.
  • HTML5: Introduced semantic elements like header, footer, article.

4. Importance of Semantic HTML

  • Improves SEO
  • Better accessibility
  • Clear structure of webpage
  • Easy maintenance

5. Common Semantic Elements

  • <header>
  • <nav>
  • <section>
  • <article>
  • <aside>
  • <footer>

6. Example Semantic Layout

</footer>
