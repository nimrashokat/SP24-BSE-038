Covered Topics

HTML Tables

Tables are used to display data in rows and columns using table tags like table, tr, th, and td.

Div Element

Div is a block level element used to divide a webpage into sections such as header, content, and footer.

Span Element

Span is an inline element used to style small parts of text inside paragraphs or headings.

Layout Design

In modern web design, div elements are preferred for creating page layouts instead of tables.

Summary
