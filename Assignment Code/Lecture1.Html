Web Technologies - Lecture 1 Summary

1. Introduction to Web Platforms

Web platforms are digital environments that allow developers and users to create, deploy and interact with web applications over the internet.

Key Components:
  • Infrastructure (Servers, Databases, Networks)
  • Development Tools (HTML, CSS, JavaScript)
  • Content Management Systems (WordPress, Drupal)
  • User Interface (Web pages, forms, navigation)
  • APIs for communication between software
  • Security systems (authentication, encryption)

2. Examples of Web Platforms

  • Social Media Platforms (Facebook, Instagram)
  • E-commerce Platforms (Shopify, Magento)
  • CMS Platforms (WordPress, Joomla)
  • Cloud Platforms (AWS, Google Cloud, Azure)

3. HTTP (Hypertext Transfer Protocol)

HTTP is the protocol used for communication between a web browser (client) and a web server.

Request Response Cycle
  • Client sends request
  • Server processes request
  • Server sends response

4. HTTP Methods

  • GET → Retrieve data
  • POST → Send data to server
  • PUT / PATCH → Update data
  • DELETE → Remove data

5. HTTP Status Codes

  • 1xx → Informational
  • 2xx → Success (200 OK)
  • 3xx → Redirection
  • 4xx → Client Error (404 Not Found)
  • 5xx → Server Error (500 Internal Error)

6. HTTPS (Secure HTTP)

HTTPS is the secure version of HTTP which uses SSL/TLS encryption to protect data between browser and server.

7. Web Standards

Web standards ensure websites work correctly across different browsers and devices.

Main Organizations:
  • W3C – World Wide Web Consortium
  • IETF – Internet Engineering Task Force
  • WHATWG – Web Hypertext Application Technology Working Group

8. HTML, CSS and JavaScript Standards

  • HTML5 – Structure of web pages
  • CSS3 – Styling and layout
  • JavaScript (ECMAScript) – Interactive behavior

9. Web Accessibility

Accessibility ensures websites are usable by people with disabilities.

Examples:
  • Screen reader support
  • Keyboard navigation
  • Proper color contrast

10. Web Constraints

Constraints are limitations developers must consider when building websites.

Types of Constraints:
  • Technical Constraints (browser compatibility, bandwidth)
  • Legal Constraints (copyright, privacy laws)
  • Design Constraints (UX, consistency)
  • Content Constraints (SEO, localization)

Example HTML Code

<!DOCTYPE html> <html> <head> <title>My First Web Page</title> </head> <body> <h1>Hello World</h1> <p>This is my first website.</p> </body> </html>

Example CSS Code

body{ background-color: lightblue; font-family: Arial; } h1{ color: darkblue; } p{ font-size:18px; }

Conclusion

Web technologies combine protocols, standards, and tools to create modern websites and web applications.
